## Summary: All Button Functions

| Button | Page | Action | API Call | Result |
|--------|------|--------|----------|--------|
| **LOGIN** | Login | Submit credentials | POST /auth/login | Stores token, goes to Dashboard |
| **REFRESH** | Dashboard | Reload data | GET /equipment, GET /maintenance | Stats + recent requests update |
| **ADD EQUIPMENT** | Equipment | Show form | None | Form appears/disappears |
| **ADD (form)** | Equipment | Create equipment | POST /equipment | New equipment in table |
| **EDIT** | Equipment | Update equipment | PUT /equipment/:id | Equipment row updates |
| **DELETE** | Equipment | Remove equipment | DELETE /equipment/:id | Equipment removed from table |
| **NEW REQUEST** | Requests | Create request | Navigate to form | Go to /requests/new |
| **FILTER (Status)** | Requests | Filter by status | None (client-side) | Table filters on change |
| **FILTER (Priority)** | Requests | Filter by priority | None (client-side) | Table filters on change |
| **FILTER (Type)** | Requests | Filter by type | None (client-side) | Table filters on change |
| **SEARCH** | Requests | Search keyword | None (client-side) | Table filters on change |
| **STATUS DROP** | Requests | Change status | PUT /maintenance/:id | Status updates, special logic applied |
| **DELETE** | Requests | Remove request | DELETE /maintenance/:id | Request removed from table |
| **EQUIPMENT SELECT** | Create Request | Choose equipment | None | Team auto-fills, preview shows |
| **SUBMIT** | Create Request | Create request | POST /maintenance | Request created, go to Kanban |
| **RESET** | Create Request | Clear form | None | Form emptied |
| **CANCEL** | Create Request | Go back | Navigate back | Return to previous page |
| **DRAG-DROP** | Kanban | Move card | PUT /maintenance/:id | Status updates, column changes |
| **PICK** | Kanban | Self-assign | POST /maintenance/:id/pick | Assigned to user, moves to In Progress |
| **PREV/NEXT** | Calendar | Change month | None | Month view changes |
| **TODAY** | Calendar | Jump to today | None | Scrolls to current date |
| **CLICK DAY** | Calendar | View events | GET /maintenance filtered | Events sidebar updates |
| **LOGOUT** | All pages | Sign out | Clear token | Return to login page |
